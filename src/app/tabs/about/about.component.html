About inbox

Inbox is an in-browser application built using Angular framework. It uses Solid server (and its Inrupt version) for data. The application's purpose is to show use of Solid technology in a user-friendly messaging application. It uses Linked Data Notifications and ActivityPub protocols for message exchange. Application takes advantage of Activity Streams as message data format.

Application was built as a diploma thesis for FIT CTU. Sources are available at GitHub.
